Bug Blitz

Bug Blitz is a first person movement FPS that involves shooting down mutated bugs with a gun while moving around the area with high speeds. Its packed with various enemies, complex movement systems, leaderboard system, and more!

Game Trailer


Features

Complex Movement Mechanics

The game is packed with various complex movement mechancis which are wall running, grappling hook, dashing, ground pound, and sliding.

AI Integration

Enemies are integrated with simple behaviour trees that provides challenge for the player.

Online Leaderboard System

A fully online leaderboard system that tracks each player's scores based on the name.
